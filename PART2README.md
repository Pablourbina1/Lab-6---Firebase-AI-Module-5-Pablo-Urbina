# Recipe Sharing
## Parte 2 del laboratorio
#### Link del video de explicación: https://drive.google.com/file/d/1YxNxf2UK4QTg5hLo53MsAJuZ2WVFGLNJ/view?usp=drive_link
En esta parte se añadio la opción de compartir una receta por medio de aplicaciones como redes sociales, gmail, entre otras, permitiendo enviar recetas a diferentes personas con
la foto generada, el nombre de la receta, y todos los ingredientes necesarios para realizarla.

## Definition of Done

| Criterio                 | Descripción                                                        |
|---------------------------|-------------------------------------------------------------------|
| Botón de compartir existe | FAB o elemento de menú que activa la opción de compartir         |
| Composable capturado      | La tarjeta de receta se renderiza como Bitmap                     |
| Imagen incluye contenido  | Imagen generada + título + ingredientes visibles                 |
| Se abre la hoja de compartir | Aparece el selector de aplicaciones nativo de Android          |
| Funciona con múltiples apps | Se puede compartir a WhatsApp, Telegram, Email, etc.           |
| Limpieza de archivos      | Los archivos temporales se eliminan después de compartir         |
| Estado de carga           | Muestra progreso mientras se prepara la imagen para compartir    |
